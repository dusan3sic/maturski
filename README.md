# Planeri procesa

### _Implementirani planeri_
- First Come First Served
- Shortest Job First
- Longest Job First

### _Odabir algoritma_
```
./pripremiZaBuild.sh [skracenica planera, npr fcfs]
```

### _Buildovanje_
- Build
```
make
```
- Clean
```
make clean
```

### _Pokretanje_
```
./run.sh
```

## Struktura projekta
- bin - bin fajlovi
- obj - object fajlovi za linkovanje projekta
- schedulers - source fajlovi za planere
